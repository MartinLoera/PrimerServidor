## Run Locally

Clone the project

```bash
  git clone https://github.com/MartinLoera/PrimerServidor
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  node app.js
```

  
## Tech Stack

**Client:** Html, text-plain

**Server:** Node.js (javascript)
